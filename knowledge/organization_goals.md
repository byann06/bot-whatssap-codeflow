# Organization Goals

## Tujuan Jangka Panjang
Code Flow Community ingin menjadi komunitas teknologi yang:
- aktif
- suportif
- berkembang bersama
- terbuka untuk semua mahasiswa

## Fokus Pengembangan
Code Flow fokus pada:
- pengembangan skill teknologi
- pengembangan soft skill
- teamwork dan kolaborasi
- project development
- budaya belajar yang sehat

## Tentang Pembelajaran
Code Flow ingin membangun sistem pembelajaran yang:
- aktif
- berkembang bersama
- tidak bergantung pada satu pemateri
- membuat semua anggota ikut berkembang

## Tentang Anggota
Code Flow ingin membantu anggota:
- lebih percaya diri
- lebih aktif
- lebih berani berbicara
- lebih siap menghadapi dunia teknologi

## Tentang Teknologi dan AI
Code Flow mendukung penggunaan teknologi dan AI secara positif untuk:
- belajar
- produktivitas
- brainstorming
- pengembangan komunitas

## Tentang Komunitas
Code Flow ingin menjadi komunitas yang:
- tidak toxic
- tidak penuh senioritas
- nyaman untuk belajar
- mendukung pemula
- mendorong kreativitas anggota

## Tentang Masa Depan Komunitas
Code Flow ingin terus:
- berkembang
- membuat sistem komunitas yang lebih baik
- membangun culture teknologi yang sehat
- membantu generasi berikutnya berkembang

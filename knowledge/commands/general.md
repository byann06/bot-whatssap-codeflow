# General Commands

## menu
Menampilkan daftar menu dan command utama bot Code Flow.

Format:
menu

## info
Menampilkan informasi data anggota yang sudah terdaftar.

Format:
info

## daftar
Digunakan untuk mendaftarkan anggota baru ke sistem Code Flow.

Format:
daftar

Atau:

daftar
nama Nama Lengkap,
npm 24042231035,
prodi Sistem Informasi,
saran Ingin aktif belajar

Catatan:
- command ini untuk anggota baru
- pengurus biasanya didaftarkan secara terpisah

## add
Digunakan untuk mengubah atau menambahkan data anggota.

Contoh:
add nama Abyan
add prodi Sistem Informasi
add saran Ingin lebih aktif

## pemateri
Menampilkan daftar pemateri kegiatan.

Format:
pemateri

## jadwalku
Menampilkan jadwal pemateri milik user.

Format:
jadwalku

## logo
Mengirim logo resmi Code Flow Community.

Format:
logo

# AI Commands

## .ai on
Mengaktifkan AI Yanverse di grup atau chat pribadi.

Format:
.ai on

Catatan:
- di grup hanya admin WhatsApp yang bisa menggunakan command ini
- jika AI aktif, Yanverse bisa mulai merespon chat

## .ai off
Menonaktifkan AI Yanverse.

Format:
.ai off

## Trigger AI di Grup
Untuk berbicara dengan Yanverse di grup, gunakan trigger:

yanverse, pesan kamu

Contoh:
yanverse, jelaskan HTML dasar
yanverse, buat contoh loop javascript

Catatan:
- tanpa trigger, Yanverse biasanya tidak membalas di grup
- sistem ini digunakan agar chat grup tidak spam AI

## AI di Private Chat
Di chat pribadi, Yanverse bisa membalas langsung tanpa trigger jika AI aktif.

Contoh:
halo yanverse
bantu jelasin CSS dong

## Tentang Yanverse
Yanverse adalah AI Assistant resmi Code Flow Community.

Yanverse membantu:
- belajar programming
- menjelaskan teknologi
- memahami command bot
- membantu pengurus dan anggota komunitas

Gaya Yanverse:
- santai
- tidak terlalu formal
- suka bercanda ringan
- tetap membantu sampai paham

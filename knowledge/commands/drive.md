# Drive & Documentation Commands

## drive auth
Digunakan admin dokumentasi untuk membuat link autentikasi Google Drive.

Format:
drive auth

## drive code
Digunakan admin dokumentasi untuk menyimpan kode autentikasi dari Google Drive.

Format:
drive code KODE_DARI_GOOGLE

Catatan:
- biasanya dilakukan sekali saat setup awal

## create folder
Digunakan untuk membuat folder dokumentasi baru di Google Drive.

Format:
create folder "nama folder"

Contoh:
create folder "pkkmb 2026"

## folder aktif
Digunakan untuk memilih folder aktif upload dokumentasi.

Format:
folder aktif "nama folder"

Contoh:
folder aktif "pkkmb 2026"

## folder list
Menampilkan daftar folder dokumentasi yang tersedia.

Format:
folder list

## remove folder
Menghapus folder dokumentasi tertentu.

Format:
remove folder "nama folder"

## upload dokumentasi
Mengupload file atau media ke Google Drive.

Format:
upload dokumentasi

Atau:
upload docs

Catatan:
- biasanya digunakan sambil mereply media/file
- file akan masuk ke folder aktif

## share to discord
Membagikan link folder dokumentasi ke Discord.

Format:
share "nama folder" to discord

Contoh:
share "pkkmb 2026" to discord

## reset
Membatalkan proses upload batch dokumentasi.

Format:
reset

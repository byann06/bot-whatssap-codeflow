# Community Values

## Belajar Bersama
Code Flow percaya bahwa belajar lebih baik dilakukan bersama daripada sendirian.

Komunitas mendorong:
- diskusi
- kolaborasi
- teamwork
- sharing knowledge

## Tidak Merendahkan Pemula
Pemula adalah bagian penting komunitas.

Tidak semua anggota langsung jago coding.

Code Flow percaya bahwa:
- semua orang pernah bingung
- semua orang pernah error
- semua orang berkembang secara bertahap

## Error Adalah Bagian Dari Belajar
Error bukan sesuatu yang memalukan.

Debugging dan kesalahan adalah bagian normal dalam belajar teknologi dan programming.

## Berkembang Bersama
Code Flow lebih fokus pada:
- pertumbuhan bersama
- saling membantu
- saling mendukung
- berkembang tanpa saling menjatuhkan

## Tidak Ada Senioritas Berlebihan
Komunitas tidak mendorong budaya senioritas toxic.

Anggota didorong untuk:
- membantu
- membimbing
- berdiskusi dengan sehat

## Teknologi Untuk Berkembang
Teknologi digunakan sebagai alat untuk:
- belajar
- berkembang
- membantu produktivitas
- membantu komunitas

## AI Sebagai Alat Bantu
AI digunakan sebagai alat bantu belajar dan produktivitas.

AI bukan pengganti proses belajar.

Anggota tetap didorong untuk:
- memahami konsep
- mencoba sendiri
- berdiskusi dengan anggota lain

## Konsisten Lebih Penting Dari Sempurna
Code Flow percaya bahwa belajar sedikit demi sedikit secara konsisten lebih baik daripada tidak belajar sama sekali.

## Berani Bertanya
Anggota didorong untuk:
- bertanya
- memberikan pendapat
- memberikan kritik dan saran
- mencoba hal baru

## Komunitas Yang Suportif
Code Flow ingin menjadi komunitas yang:
- santai
- suportif
- terbuka
- aktif
- berkembang bersama

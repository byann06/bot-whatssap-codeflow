# AD/ART Code Flow Community

## Tentang Komunitas
Code Flow Community (CFC) adalah Unit Kegiatan Mahasiswa Universitas Merangin yang berfokus pada pembelajaran teknologi, coding, dan pengembangan skill digital.

UKM ini berdiri sejak tahun 2025.

## Prinsip Komunitas
Code Flow memiliki prinsip:
- kekeluargaan
- kemandirian
- transparansi
- akuntabilitas

## Visi
Menjadi wadah pembelajaran coding bagi seluruh mahasiswa lintas prodi serta tumbuh bersama secara suportif dan kolaboratif.

## Misi
- menjadi wadah pembelajaran coding
- membangun budaya kolaboratif
- membangun komunitas aktif dan inovatif
- meningkatkan semangat belajar mandiri dan teamwork
- membantu anggota mengembangkan potensi teknologi

## Tujuan Komunitas
- meningkatkan kemampuan mahasiswa di bidang pemrograman
- mewadahi pengembangan software dan web
- mendorong inovasi teknologi di lingkungan kampus

## Keanggotaan
Code Flow terbuka untuk:
- mahasiswa aktif Universitas Merangin
- seluruh prodi
- anggota yang memiliki minat belajar teknologi dan programming

## Hak Anggota
Anggota berhak:
- mengikuti kegiatan komunitas
- memberikan pendapat dan saran
- mendapatkan pembelajaran dan pengembangan skill

## Kewajiban Anggota
Anggota wajib:
- menjaga nama baik komunitas
- mengikuti aturan komunitas
- menjaga kerahasiaan dokumen atau materi internal
- aktif berpartisipasi dalam kegiatan

## Budaya Komunitas
Code Flow memiliki budaya:
- belajar bersama
- saling membantu
- tidak toxic
- suportif terhadap pemula
- berkembang bersama tanpa saling menjatuhkan

## Struktur Pengurus
Struktur utama komunitas:
- Pembina
- Ketua
- Wakil Ketua
- Sekretaris
- Bendahara
- Divisi Media Edukasi Digital
- Divisi Perlengkapan dan Logistik

## Masa Jabatan
Masa kepengurusan berlangsung selama:
- 2 semester
- atau 1 tahun

## Tentang Yanverse
Yanverse adalah AI Assistant resmi Code Flow Community.

Yanverse membantu:
- pembelajaran teknologi
- memahami command bot
- membantu anggota dan pengurus
- membantu dokumentasi dan organisasi komunitas

# Project Guidelines

## Tentang Project
Code Flow mendorong anggota untuk membuat project sebagai bagian dari proses belajar.

Project dianggap sebagai:
- media belajar
- latihan teamwork
- latihan problem solving
- latihan workflow development

## Jenis Project
Contoh project:
- website
- bot
- UI/UX design
- automation
- AI project
- komunitas tools

## Teamwork
Saat membuat project bersama:
- anggota didorong aktif berdiskusi
- saling membantu
- saling review
- tidak membebankan semua tugas ke satu orang

## GitHub
Anggota didorong untuk:
- menggunakan GitHub
- commit project secara rutin
- belajar version control
- menyimpan portfolio project

## Tentang Error
Saat membuat project:
- error dianggap normal
- debugging adalah bagian dari proses belajar
- anggota tidak perlu takut mencoba

## Filosofi Project
Code Flow lebih menghargai:
- progress kecil tapi konsisten
daripada:
- banyak rencana tanpa praktik

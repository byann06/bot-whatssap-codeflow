# Internal Culture Code Flow Community

## Cara Komunitas Belajar
Code Flow lebih suka belajar dengan cara santai dan pelan-pelan.

Anggota tidak dituntut langsung jago.

Fokus utama:
- konsisten belajar
- saling bantu
- praktik langsung
- berkembang bersama

## Budaya Komunitas
Budaya komunitas Code Flow:
- santai tapi tetap serius belajar
- suka bercanda dan tongkrongan
- tidak terlalu formal
- tidak merendahkan pemula
- lebih suka membimbing daripada menghakimi

## Tentang Pemula
Pemula dianggap sebagai bagian penting komunitas.

Tidak masalah:
- belum bisa coding
- masih bingung syntax
- sering error
- masih belajar dasar

Yang penting:
- mau belajar
- mau mencoba
- mau berkembang

## Cara Yanverse Berbicara
Yanverse berbicara:
- santai
- seperti teman tongkrongan teknologi
- suka bercanda ringan
- suka jokes dan sarkas ringan
- tetap membantu sampai paham

Yanverse tidak:
- toxic berlebihan
- terlalu formal
- terlalu corporate
- terlalu banyak emoji
- sok paling pintar

## Filosofi Komunitas
Code Flow percaya bahwa:
- semua orang bisa belajar teknologi
- error adalah bagian dari belajar
- skill berkembang secara bertahap
- komunitas harus jadi tempat berkembang bersama

## Tentang Error dan Debugging
Di Code Flow, error bukan hal memalukan.

Error dianggap bagian normal dari proses belajar programming.

## Tentang Project
Project dianggap sebagai cara belajar terbaik.

Anggota didorong untuk:
- mencoba membuat project
- belajar teamwork
- belajar problem solving
- belajar dari kesalahan

# About Code Flow Community

Code Flow Community adalah komunitas belajar teknologi dan programming.

Komunitas ini dibuat untuk:
- belajar coding bersama dan berkembang bersama
- belajar bareng tanpa takut pusing sendiri
- mentoring anggota
- membuat project bersama
- membangun relasi dan teamwork

Code Flow terbuka untuk:
- pemula di dunia programming
- mahasiswa
- anggota yang ingin belajar teknologi
- anggota yang ingin berkembang bersama komunitas

Fokus pembelajaran di Code Flow:
- Frontend Development
- Backend Development
- UI/UX
- Artificial Intelligence
- Editing & Media
- Project Collaboration

Budaya komunitas:
- santai tapi tetap belajar
- saling membantu
- tidak toxic
- tidak merendahkan pemula
- belajar pelan-pelan sampai paham
- berkembang bersama tanpa saling menjatuhkan

Yanverse adalah AI Assistant resmi Code Flow Community yang membantu anggota memahami command bot, kegiatan komunitas, dan pembelajaran teknologi.

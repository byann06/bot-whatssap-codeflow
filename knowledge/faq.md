# Frequently Asked Questions

## Apakah pemula boleh masuk Code Flow?
Boleh banget.

Code Flow terbuka untuk pemula yang ingin belajar teknologi dan programming dari dasar.

## Harus jago coding dulu?
Tidak harus.

Code Flow dibuat untuk belajar bersama pelan-pelan sampai paham.

## Kalau belum ngerti materi gimana?
Boleh bertanya ke anggota lain, mentor, pengurus, atau Yanverse.

Tidak perlu takut dianggap noob.

## Yanverse itu apa?
Yanverse adalah AI Assistant resmi Code Flow Community.

Yanverse membantu:
- menjelaskan materi
- membantu command bot
- membantu anggota memahami teknologi
- membantu pengurus komunitas

## Kenapa Yanverse tidak membalas di grup?
Biasanya karena:
- AI belum diaktifkan
- tidak menggunakan trigger

Gunakan:
yanverse, pertanyaan kamu

Contoh:
yanverse, jelaskan JavaScript dasar

## Cara mengaktifkan AI?
Gunakan:

.ai on

Catatan:
- di grup hanya admin WhatsApp yang bisa mengaktifkan AI

## Kenapa command hadir tidak bisa dipakai?
Biasanya karena sesi absensi belum dibuka admin.

## Apakah Code Flow hanya fokus coding?
Tidak.

Code Flow juga belajar:
- teamwork
- project collaboration
- media & editing
- AI
- organisasi komunitas

## Apakah boleh bertanya basic?
Boleh.

Code Flow dibuat untuk belajar bersama tanpa merendahkan pemula.

# Communication Style

## Cara Komunikasi Komunitas
Code Flow lebih nyaman menggunakan komunikasi:
- santai
- jelas
- tidak terlalu formal
- tetap sopan
- mudah dipahami

## Saat Berdiskusi
Anggota didorong untuk:
- menghargai pendapat orang lain
- tidak toxic
- tidak merendahkan pemula
- fokus pada solusi

## Cara Yanverse Berbicara
Yanverse berbicara seperti:
- teman tongkrongan teknologi
- mentor santai
- anggota komunitas

Yanverse:
- boleh bercanda ringan
- boleh sarkas ringan
- tetap membantu
- tidak terlalu toxic
- tidak terlalu formal

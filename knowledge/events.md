# Activities & Events

## Kegiatan Rutin
Code Flow memiliki beberapa kegiatan rutin seperti:
- belajar coding bersama
- mentoring
- diskusi teknologi
- sharing project
- workshop komunitas

## Mentoring
Mentoring dilakukan untuk membantu anggota memahami materi teknologi dan programming secara pelan-pelan.

Tujuan mentoring:
- membantu pemula memahami dasar teknologi
- membimbing anggota dalam praktik coding
- membantu anggota berkembang bersama komunitas

## Workshop
Workshop biasanya membahas:
- web development
- frontend
- backend
- AI
- UI/UX
- editing dan media

## Project Collaboration
Anggota dapat membuat project bersama untuk:
- belajar teamwork
- meningkatkan skill
- membangun portofolio
- belajar workflow project nyata

## Dokumentasi Kegiatan
Setiap kegiatan biasanya memiliki dokumentasi yang disimpan oleh tim dokumentasi komunitas.

Dokumentasi dapat berupa:
- foto kegiatan
- video kegiatan
- file materi
- arsip komunitas

## Budaya Belajar
Code Flow lebih fokus pada:
- belajar bersama
- praktik langsung
- berkembang bersama
- membantu pemula
- suasana komunitas yang santai dan suportif

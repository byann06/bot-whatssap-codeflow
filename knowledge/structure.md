# Structure of Code Flow Community

## Pembina
Pembina bertugas:
- membimbing komunitas
- mengarahkan pengembangan UKM
- membantu evaluasi kegiatan
- mendukung perkembangan anggota dan pengurus

## Ketua
Ketua bertugas:
- memimpin komunitas
- mengawasi program kerja
- mengambil keputusan utama komunitas
- menjaga arah dan perkembangan komunitas

## Wakil Ketua
Wakil Ketua bertugas:
- membantu ketua
- menggantikan ketua jika berhalangan
- membantu koordinasi internal komunitas

## Sekretaris
Sekretaris bertugas:
- mengelola administrasi komunitas
- membuat notulen rapat
- mengatur surat dan dokumen
- membantu koordinasi kegiatan

## Bendahara
Bendahara bertugas:
- mengelola keuangan komunitas
- mencatat pemasukan dan pengeluaran
- membuat laporan keuangan
- menjaga transparansi keuangan

## Media Edukasi Digital (MEDIG)
MEDIG bertugas:
- membuat desain komunitas
- dokumentasi kegiatan
- mengelola sosial media
- membuat konten edukasi

## Perlengkapan dan Logistik (PERLOG)
PERLOG bertugas:
- menyiapkan perlengkapan kegiatan
- mengatur kebutuhan acara
- menjaga inventaris komunitas
- membantu teknis kegiatan

# Divisions in Code Flow Community

## Media Edukasi Digital (MEDIG)
Divisi MEDIG bertugas:
- membuat desain komunitas
- mengelola sosial media
- membuat dokumentasi kegiatan
- membuat konten edukasi
- membantu branding komunitas

Skill yang cocok:
- desain grafis
- editing video
- content creator
- sosial media
- kreativitas visual

## Perlengkapan dan Logistik (PERLOG)
Divisi PERLOG bertugas:
- menyiapkan perlengkapan kegiatan
- mengatur kebutuhan acara
- membantu koordinasi teknis kegiatan
- menjaga inventaris komunitas

Skill yang cocok:
- teamwork
- koordinasi
- komunikasi
- manajemen perlengkapan
